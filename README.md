# My Web App

A modern web application with a **frontend** and **admin panel**, built using **React**, **TypeScript**, and **Vite**, with **Firebase** for backend services.

The frontend provides a seamless user experience, while the admin panel enables management of products, users, and orders.

---

## Live Demo

[View the live site](https://gleeful-alpaca-d2f098.netlify.app/)

---

## Features

* Responsive and interactive UI
* Product, cart, and wishlist management
* User authentication and profile management
* Admin panel with product, category, and user management
